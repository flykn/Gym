# Gym
# 🏋️ GymTracker — PWA Дневник тренировок

Progressive Web App для учёта тренажеров и ведения дневника тренировок.

## Возможности
- ➕ Добавление тренажеров (название, фото, повторения, вес)
- 📷 Загрузка фото через файл (base64) или URL
- 📅 Дневник тренировок с группировкой по дням
- 💾 Хранение данных в localStorage
- 📱 Работает офлайн (Service Worker)
- ⬇️ Устанавливается как приложение

## Запуск локально
Из-за Service Worker нужен HTTP-сервер:
```bash
python -m http.server 8080
# или
npx serve
